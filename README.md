# Mining Data Analytics ⛏️💎

This repository focuses on data cleaning, exploratory analysis, and statistical evaluation of mineral exploration and geochemical datasets.

## Project Scope
- **Data Preparation**: Cleaning raw assay results, handling missing values, and unit normalization.
- **Statistical Analysis**: Evaluating measures of center (Mean, Median) and spread (Standard Deviation, IQR).
- **Correlation Studies**: Using Pearson coefficients to find geochemical associations (e.g., Gold-Silver correlation).
- **Visualization**: Creating professional geological charts and histograms.

## Exercises
